关于ssh登录机制

配置密钥与私钥

生成密钥对

ssh-keygen -t rsa -b 4096 -f ~/.ssh

部署公钥

ssh-copy-id -i ~/.ssh/rsa_id.pub user@server

完成后公钥会自动添加到 ~/.ssh/authorized_keys

也可以手动配置公钥

直接将公钥内容写到authorized_keys里面就可以了

简单来说公钥就是关门的（锁） 私钥就是开门的（钥匙）
